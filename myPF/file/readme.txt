    <h1>GUIDE</h1>
    <hr>
    <br>
    <P>1. OUTPUT창에 제일 오른쪽 버튼을 클릭하여 새로운 문서를 만듭니다.</P>
    <P>2. 화면 제일 아래 입력란에 HTML 태그를 입력하고 오른쪽 입력 버튼을 클릭합니다.</P>
    <P>3. 입력한 태그가 OUTPUT창에 출력됩니다.</P>
    <P>4. OUTPUT창에 입력된 모든 태그들을 복사하고 싶다면 OUTPUT창에 오른쪽 두번째 버튼을 클릭해 클립보드에 복사합니다.</P>
    <P>5. 문서를 서버에 저장하고 싶다면 OUPUT창에 오른쪽 세번째 버튼을 클릭해 파일 이름을 입력후 저장합니다.</P>
    <P>6. 문서의 내용을 모두 지우고 싶다면 OUPUT창에 제일 왼쪽 버튼을 클릭해 지웁니다.</P>
    <P>7. 화면 왼쪽 파일 목록에서 서버에 저장되어 있는 파일을 불러 올 수 있습니다.</P>
